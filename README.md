# Student Score Predictor

An end-to-end machine learning project that predicts students' writing scores based on demographic, educational, and academic performance data. The project implements a complete ML pipeline including data ingestion, data transformation, model training, evaluation, and deployment preparation using industry-standard machine learning engineering practices.

### Dataset Features

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score

### Target Variable

* Writing Score
